# Лабораторная работа №5: Jetpack Compose - Контейнеры, ARGB и Material Design
## Цель работы:
Повторить основы разработки пользовательского интерфейса на Jetpack Compose, работу с контейнерами Row и Column, познакомиться с ARGB-моделью цветов, компонентом Scaffold и принципами Material Design.
## Описание приложения
Разработано простое приложение, демонстрирующее основные принципы построения интерфейсов в Jetpack Compose. Архитектура приложения построена на модульных Composable-функциях.
- **`MainScreen()`** — главный экран, собирающий все компоненты 
- **`TitleBlock()`** — заголовок из двух строк текста
- **`ColorRow()`** — горизонтальный ряд из трёх цветных квадратов (с использованием `Row`)
- **`ColorBox(color: Color)`** — переиспользуемый компонент для отображения цветного квадрата
- **`CombinedBlock()`** — комбинированный блок с вложенными `Row` и `Column`

Цвета заданы в **ARGB-формате**
